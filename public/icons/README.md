# Icon Assets

Place your favicon and app icons in this directory with the following specifications:

## Required Icons:
- `icon-72x72.png` - 72x72 pixels
- `icon-96x96.png` - 96x96 pixels
- `icon-128x128.png` - 128x128 pixels
- `icon-144x144.png` - 144x144 pixels
- `icon-152x152.png` - 152x152 pixels
- `icon-192x192.png` - 192x192 pixels (recommended size for Android)
- `icon-384x384.png` - 384x384 pixels
- `icon-512x512.png` - 512x512 pixels (recommended size for app stores)

## Favicon:
- `favicon.ico` - 48x48 pixels (or multiple sizes in one ICO file)

These icons are referenced in the `manifest.json` file for Progressive Web App (PWA) functionality and proper display across different devices and platforms.
