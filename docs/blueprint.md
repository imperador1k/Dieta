# **App Name**: DietaS: Your Data-Aesthetic Diet Companion

## Core Features:

- Interactive Dashboard: Visually appealing and animated dashboard providing a modular overview of daily diet metrics with smooth transitions and animations.
- Personalized Diet Architect: Organized interface for setting and adjusting global dietary goals (calories, macros) and defining micronutrient targets with an elegant, searchable list.
- AI-Powered Food Logger: AI tool predicts and suggests food entries based on user input. Simplifies daily food logging using natural language processing and learns user patterns for quicker entry.
- Barcode Scanner Integration: Scan product barcodes using the device camera to automatically fetch nutritional information from APIs (OpenFoodFacts, USDA) for quick food logging.
- Evolution Gallery: Visually track progress with metric graphs (weight, fat, muscle, calorie intake) and a photo gallery that stores dates and metrics. Includes side-by-side comparison mode.
- Local Data Storage with Optional Sync: Minimalist back-end with local data storage with optional synchronization to Firebase or Supabase to persist data and provide optional sync features across devices. Tables for UserGoals, DailyLog, BodyMetrics, CustomFoods and Photos.

## Style Guidelines:

- Primary color: Deep violet (#483D8B), lending sophistication and visual intrigue.
- Background color: Very light purple (#F0F8FF) creating a subtle backdrop that does not detract from more colorful UI elements.
- Accent color: A contrasting indigo (#4B0082), used sparingly to draw the eye to key interactions and CTAs.
- Body and headline font: 'Alegreya', a humanist serif offering both elegance and readability.
- iOS SF Symbols, or equivalent, offering consistency and clarity throughout the UI.
- Neumorphism with soft shadows and glassmorphism effects on widgets to enhance visual depth and interactivity.
- Organic, iOS-style spring animations for fluid transitions and micro-interactions, enriching the user experience.
